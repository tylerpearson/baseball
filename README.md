# Pitching 101

The complete beginner's guide to MLB pitching. A single-page interactive site covering pitch types, movement, strategy, and the legends who mastered the craft.

## Sections

- **Basics** -- How an at-bat works, balls, strikes, counts, and the fundamental rules
- **Cheat Sheet** -- Quick-reference card for common pitch types at a glance
- **Pitch Arsenal** -- Deep dive into 10 pitch types (four-seam, slider, curveball, changeup, etc.) with animated trajectory diagrams, velocity/break stats, and usage context
- **Pitcher Types** -- Comparison of power pitchers vs. finesse pitchers
- **Strategy** -- Count matrix showing pitcher/batter advantage, interactive strike zone heat maps with per-pitch MLB usage stats
- **Compare** -- Side-by-side pitch comparison tool
- **Legends** -- Profiles of all-time great pitchers
- **Glossary** -- Searchable glossary with inline tooltip support

## Tech

Static single-page site. No frameworks, no build step.

- `index.html` -- all content
- `styles.css` -- dark theme using CSS custom properties
- `scripts.js` -- interactivity (tooltips, pitch comparisons, animated SVG trajectories, strike zone heat maps)

## Run locally

Open `index.html` in a browser.
